# Apps

Esta pasta concentra as superficies executaveis do produto:

- `api`: backend central de negocio
- `web-client`: vitrine, catalogo, checkout e tracking
- `web-partner`: cockpit da adega
- `mobile`: reservado para V2/V3

Regra: a logica de negocio fica centralizada na `api`, nao espalhada nos frontends.
