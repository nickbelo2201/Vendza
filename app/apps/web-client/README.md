# Web Client

Superficie publica do produto:

- vitrine
- busca
- produto
- carrinho
- checkout
- tracking

Prioridade: conversao, clareza e velocidade mobile-first.

## Status atual

Esta aplicacao ja esta estruturada para o MVP com as rotas base do cliente:

- `/`
- `/produto/[slug]`
- `/checkout`
- `/pedidos/[publicId]`

O layout ainda nao e o visual final. O projeto segue bloqueado pelo `design gate`, entao a prioridade aqui e integracao, estado e fluxo de compra ponta a ponta.
